These scenes demonstrate how to save and load game data in various ways.

1. The basic demo shows a very simple method of storing data with Unity's PlayerPrefs class.
2. In the second example, we show how to store and load more complex data using Unity's PlayerPrefs class.
3. In the third example, we introduce JSON to allow for much more data to be easily stored and loaded.
4. Finally, we move on from the PlayerPrefs class and begin saving data using more reliable methods.
